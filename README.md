# 🚀 Team Task Manager (Live Project)

A full-stack Task Management Web Application built using FastAPI, SQLAlchemy, and HTML/CSS/JS.

## 🌐 Live Demo
https://observant-heart-production-92f3.up.railway.app/login_page
## 📌 Features
- User Authentication (Admin & Member)
- Role-based Access Control
- Project Management
- Task Creation & Assignment
- Dashboard with Charts

## 🛠️ Tech Stack
- FastAPI
- SQLAlchemy
- SQLite
- HTML/CSS/JS
- Chart.js

## ▶️ Run Locally
uvicorn main:app --reload
